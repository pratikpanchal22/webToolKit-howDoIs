HTML5 Starter template

This is a basic boilerplate template to start with html5 with placeholders for css & js files

  1. Create index.html

    1. Declarations

        2. <!doctype html>
        <html lang="en">
           <!--Head goes in here-->
           <!--Body goes in here-->
        </html>

    2. Head

        • Character encoding: meta charset
        • Title
        • Description
        • Author
        • Viewport
        • Stylesheets

        <head>
           <meta charset="utf-8">
           <title>HTML5 Starter template</title>
           <meta name="description" content="HTML Starter template">
           <meta name="author" content="Pratik Panchal">
           <link rel="stylesheet" href="css/styles.css">
        </head>

    3. Body

        • Link javascript here

        <body>
           <script src="js/scripts.js"></script>
           <!-- content goes in here -->
        </body>

  2. Create scripts directory

    1. Create js directory at same location as index.html and intialize it with scripts.js file

  3. Create styles directory

    1. Create css directory at same location as index.html and intialize it with styles.css file
